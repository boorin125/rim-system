## 🚀 เริ่มต้นอย่างรวดเร็ว (5 นาที!)

### ขั้นที่ 1: คัดลอกไฟล์ (3 ไฟล์)
```bash
cd D:\Projects\RIM-System\backend

copy [ดาวน์โหลด]\.gitignore .
copy [ดาวน์โหลด]\README.md .
copy [ดาวน์โหลด]\.env.example .
```

### ขั้นที่ 2: เริ่ม Git
```bash
git init
git config --global user.name "ชื่อคุณ"
git config --global user.email "email@example.com"
```

### ขั้นที่ 3: Commit
```bash
git add .
git commit -m "เริ่มต้นโปรเจค RIM System"
```

### ขั้นที่ 4: อัพโหลด
```bash
# สร้าง repo บน GitHub.com ก่อน
git remote add origin https://github.com/[user]/rim-system.git
git push -u origin main
```

**เสร็จแล้ว!** ✅

## 🔑 GitHub Personal Access Token

### ต้องมี! (ไม่ใช้รหัสผ่านปกติ)

**วิธีสร้าง:**
1. GitHub.com → Settings
2. Developer settings → Personal access tokens
3. Generate new token (classic)
4. ติ๊ก "repo"
5. คัดลอก Token ทันที!

**วิธีใช้:**
```
Username: [ชื่อ GitHub]
Password: [วาง Token ที่คัดลอก]
```

## 🔍 เช็คอย่างรวดเร็ว

### ก่อน Commit:
```bash
git status          # มีอะไรเปลี่ยน?
git status | find ".env"   # .env อยู่ไหม? (ต้องไม่มี!)
git diff            # เปลี่ยนอะไรบ้าง?
```

### Commit:
```bash
git add .
git commit -m "feat: ทำอะไร"
```

### อัพโหลด:
```bash
git pull            # ดึงล่าสุดก่อน
git push            # ส่งขึ้น GitHub
```

## 🚨 แก้ฉุกเฉิน

### ถ้า commit .env ไปแล้ว:
